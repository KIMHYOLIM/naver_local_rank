🏥 네이버 지역 순위 체크 시스템 - 실무용 가이드

📦 설치 방법 (5분 완료)
====================================
1. "실무용_설치.bat" 더블클릭
2. 화면 안내 따라하기
3. 네이버 API 키 입력 (아래 참고)
4. 완료!

🔑 네이버 API 키 발급 방법
====================================
1. https://developers.naver.com 접속
2. 네이버 아이디로 로그인
3. "Application 등록" 클릭
4. 앱 이름: 아무거나 (예: 순위체크)
5. 사용 API: "검색" 체크
6. 환경: "WEB" 선택
7. 서비스 URL: http://localhost (아무거나)
8. Client ID와 Client Secret 복사해서 입력

📋 사용 방법 (매우 쉬움)
====================================
1. 바탕화면 "순위체크.bat" 더블클릭
2. 메뉴에서 번호 선택:
   - 1: 빠른 순위 확인
   - 2: 상세 리포트
   - 3: Excel 파일 열기
   - 4: 새 데이터 수집
   - 5: 웹 대시보드 (예쁨!)

🔧 내 업체 정보 설정
====================================
"keywords.csv" 파일을 메모장으로 열어서:
- keyword: 검색할 키워드 (예: "강남 카페")
- branch: 지점명 (예: "강남점")  
- target_patterns: 찾을 업체명 (예: "스타벅스 강남|강남 스타벅스")

📊 결과 해석
====================================
- 1위~5위: 매우 좋음! 🥇
- 6위~10위: 좋음 📈
- 11위~20위: 보통 📊
- 순위없음: 개선 필요 ⚠️

🚨 문제 해결
====================================
- "API 오류": 네이버 API 키 확인
- "파일 없음": 폴더 위치 확인
- "Python 없음": Python 설치
- 기타: 구글에 "네이버 API 오류" 검색

💡 활용 팁
====================================
- 매일 체크해서 순위 변화 모니터링
- Excel 파일로 차트 만들어서 보고서 작성
- 웹 대시보드로 팀원들과 공유
- 경쟁업체 순위도 함께 모니터링
